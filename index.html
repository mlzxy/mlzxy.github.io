  • Oct 2017 »
    [Project] Binary Network Training and Inference Library

    In this project, I implemented a library that supports training binary network and export models for inference. The library has an easy-to-use python interface through MxNet, archives state of the art in many binary network benchmarks. But they are all irrelevant.

    The real gem is that I design a special GEMM kernel supporting binary matrix multiplication with arbitrary bits, following Anatomy of High-Performance Matrix Multiplication.

    It involves several layers of tricky tiling for cache reusability, writing assembly in critical sections, SIMD, efficient binary data format, etc. It is a DAUNTING ONE, but I somehow pull it off. 😂

    I usually don't put company projects here. But this one has a clear open-source baseline and outperforms it by order of magnitude. I am proud of this one.

    Because of the corporative privacy policy, I cannot disclose detailed information of this project, but welcome to discuss related topics with me :-)
